> - OpenAI 兼容接口 `POST /v1/files`，使用 `multipart/form-data`
> - 上传本地文件后返回 `file_id`
> - 在 [Chat Completions](/zh/api-manual/language-series/openai/create-chat-completion) 或 [Responses API](/zh/api-manual/language-series/openai/openai-responses) 中引用
> - 也可在消息中直接使用 URL 或 Base64，见[多模态指南](/zh/guide/advanced/multimodal)

## purpose 取值

| `purpose` | 典型用途 |
| --- | --- |
| `assistants` | Assistant / 工具工作流 |
| `vision` | 视觉模型图片 |
| `user_data` | 用户文档 |
| `batch` | Batch API 输入 |
| `fine-tune` | 微调数据集 |

## 示例（cURL）

```bash
curl https://api.shuyou.ai/v1/files \
  -H "Authorization: Bearer $ShuYou_API_KEY" \
  -F purpose="assistants" \
  -F file="@document.pdf"
```

将返回的 `id` 作为后续请求中的 `file_id` 使用。
