---
title: 创建预测任务
description: 提交异步 Prediction 任务。
openapi: zh/api-manual/task-management/create-prediction.json POST /v1/predictions
---

> - 用于第三方/托管模型的异步任务
> - 返回 `prediction_id`，通过[查询预测任务](/zh/api-manual/task-management/get-prediction)轮询
> - `input` 结构因模型而异，见[模型目录](https://shuyou.ai/models)

<Steps>
  <Step title="创建任务">
    `POST /v1/predictions`，传入 `model` 与 `input`。
  </Step>
  <Step title="轮询状态">
    `GET /v1/predictions/{prediction_id}` 直至 `status` 为 `succeeded` 或 `failed`。
  </Step>
  <Step title="使用结果">
    从 `output` 读取生成资源 URL 或结构化数据。
  </Step>
</Steps>
