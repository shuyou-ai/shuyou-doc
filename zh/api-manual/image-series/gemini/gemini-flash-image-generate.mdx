> - `google/gemini-2.5-flash-image` 支持文生图与图生图
> - 兼容 OpenAI Images API，可将 OpenAI SDK 的 `base_url` 设为 `https://api.shuyou.ai/v1`
> - Vertex 原生多模态调用见[图像生成指南](/zh/guide/advanced/image-generation)
> - 请及时保存生成的图片 URL；可用性取决于上游服务商策略

<Tip title="相关模型">
还可使用 `google/gemini-3-pro-image-preview`、`google/gemini-2.5-flash-image-free` 等。浏览[模型目录](https://shuyou.ai/models)。
</Tip>

## 认证方式

所有请求需携带 Bearer Token：

```http
Authorization: Bearer YOUR_API_KEY
```

在 [ShuYou 控制台](https://shuyou.ai/signin) 创建 API Key。

## 调用流程

<Steps>
  <Step title="发起生成请求">
    调用 `POST /v1/images/generations`，传入 `model` 与 `prompt`。
  </Step>
  <Step title="获取图片结果">
    响应中的 `data[].url` 为生成图片地址（或配置 `b64_json` 时返回 Base64）。
  </Step>
  <Step title="可选：查询用量">
    按量付费密钥可通过[查询生成记录](/zh/api-manual/task-management/get-generation)查看计费信息。
  </Step>
</Steps>
