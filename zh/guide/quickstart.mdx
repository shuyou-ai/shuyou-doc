# 快速开始

欢迎使用 **[AGIPower](https://agipower.ai/zh/about/intro.html)**！本指南将帮助你快速上手，从注册到发起第一个 API 请求，一步步引导你使用 AGIPower 的全部能力。

<Tip title="💡 四步即可开始">
只需要四个简单步骤，即可开始使用 AGIPower：
</Tip>

1. **登录 AGIPower**：访问 **[AGIPower 登录页面](https://agipower.ai/signin)**，选择以下任一方式登录：

   - Google 账号登录
   - GitHub 账号登录
   - 微信登录
   

2. **选择使用方式**：根据你的使用场景，选择 [Pay As You Go（按量付费）](#选择使用方式) 或 [Builder Plan（订阅制）](#选择使用方式)。

3. **获取 API 密钥**：登录后，在对应的管理页面创建 API Key。

4. **发起你的第一个请求**：选择你熟悉的 [API 协议](#支持的-api-协议)，复制下面的代码示例，替换你的 API Key，即可运行。

---

## 选择使用方式

AGIPower 提供两种使用方式，适配不同的使用场景：

### Pay As You Go（按量付费）

**适用场景**：生产环境、商业化产品、高并发应用

Pay As You Go 采用**预充值 + 按实际消耗扣费**的模式，专为生产环境打造：

- **无 Rate Limit 限制**，支持高并发调用
- **按 Token 精确计费**，成本透明可控
-

前往 **[Pay As You Go 管理页面](https://agipower.ai/platform/pay-as-you-go)** 充值并创建 API Key。

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/01/19/rsGoojf/pay-as-you-go.png" alt="Pay As You Go 管理页面" />

<Info title="📚 详细指南">
完整的使用说明请参考 [Pay As You Go 使用指南](/zh/guide/pay-as-you-go)。
</Info>

### Builder Plan（订阅制）

**适用场景**：个人开发、学习探索、Vibe Coding、快速原型

Builder Plan 提供固定月费、可预测的 AI 模型调用方案：

- **固定月费**，$20/月起
- **全明星模型矩阵**，一个订阅调度全球 100+ 顶尖模型
- **全场景覆盖**，Coding + 生图 + 生视频 + 对话一站式搞定

前往 **[Pricing 页面](https://agipower.ai/pricing/subscription)** 了解套餐详情并订阅。

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/01/28/iszJOqe/ss-1.png" alt="订阅制套餐对比" />

<Info title="📚 详细指南">
完整的使用说明请参考 [订阅制套餐指南](/zh/guide/subscription)。
</Info>

### 两种方式对比

| 对比维度 | Pay As You Go（按量付费） | Builder Plan（订阅制） |
|---------|--------------------------|----------------------|
| **适用场景** | 生产环境、商业化产品 | 个人开发、学习探索 |
| **计费方式** | 按实际消耗精准计费 | 固定月费 |
| **Rate Limit** | 无限制 | 10-15 RPM |
| **并发限制** | 无限制 | 有 Weekly Limit |
| **使用限制** | 无特殊限制 | 禁止生产环境使用 |

<Warning title="⚠️ 重要提示">
如果你的项目已经上线或即将商业化，**必须使用 Pay As You Go 方案**。订阅制仅限个人开发和学习场景，禁止用于生产环境。
</Warning>

## 支持的 API 协议

AGIPower 支持四种主流 API 协议，你可以使用最熟悉的 SDK 调用 AGIPower 上的所有模型：

| 协议 | Base URL | 兼容 SDK | 说明 |
|------|----------|----------|------|
| **OpenAI Chat Completions** | `https://api.agipower.ai/v1` | OpenAI SDK | 最广泛使用的 Chat API |
| **OpenAI Responses** | `https://api.agipower.ai/v1` | OpenAI SDK | OpenAI 新一代 Responses API |
| **Anthropic Messages** | `https://agipower.ai` | Anthropic SDK | Claude 系列原生协议 |
| **Google Gemini** | `https://api.agipower.ai` | Google GenAI SDK | Gemini 系列原生协议 |

<Tip title="💡 跨协议调用">
AGIPower 的核心优势之一是**协议无关性**——你可以通过任意支持的协议调用任意模型。例如，用 OpenAI SDK 调用 Claude 模型，或用 Anthropic SDK 调用 Gemini 模型。
</Tip>

### 协议一：OpenAI Chat Completions

<Info title="兼容性说明">
AGIPower 的 API 端点与 OpenAI Chat Completions API 完全兼容，只需修改 `base_url` 和 `api_key` 两个参数即可无缝切换。
</Info>



```python title="Python"
from openai import OpenAI

# 1. 初始化 OpenAI 客户端
client = OpenAI(
    # 2. 将基础 URL 指向 AGIPower 端点
    base_url="https://api.agipower.ai/v1", # [!code highlight]
    # 3. 替换为你从 AGIPower 获取的 API Key
    api_key="<你的 AGIPower_API_KEY>", # [!code highlight]
)

# 4. 发起 Chat Completion 请求
completion = client.chat.completions.create(
    # 5. 指定模型，格式为 "供应商/模型名称"
    model="openai/gpt-5", # [!code highlight]
    messages=[
        {
            "role": "user",
            "content": "生命的意义是什么？"
        }
    ]
)

print(completion.choices[0].message.content)
```

```ts title="TypeScript"
import OpenAI from "openai";

// 1. 初始化 OpenAI 客户端
const openai = new OpenAI({
  // 2. 将基础 URL 指向 AGIPower 端点
  baseURL: "https://api.agipower.ai/v1", // [!code highlight]
  // 3. 替换为你从 AGIPower 获取的 API Key
  apiKey: "<你的 AGIPower_API_KEY>", // [!code highlight]
});

async function main() {
  // 4. 发起 Chat Completion 请求
  const completion = await openai.chat.completions.create({
    // 5. 指定模型，格式为 "供应商/模型名称"
    model: "openai/gpt-5", // [!code highlight]
    messages: [
      {
        role: "user",
        content: "生命的意义是什么？",
      },
    ],
  });

  console.log(completion.choices[0].message.content);
}

main();
```

```bash title="cURL"
curl https://api.agipower.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $AGIPower_API_KEY" \
  -d '{
    "model": "openai/gpt-5",
    "messages": [
      {
        "role": "user",
        "content": "生命的意义是什么？"
      }
    ]
  }'
```

### 协议二：OpenAI Responses

<Info title="兼容性说明">
AGIPower 同样兼容 OpenAI 新一代 Responses API。Base URL 与 Chat Completions 相同，调用方式更加简洁。
</Info>



```python title="Python"
from openai import OpenAI

client = OpenAI(
    base_url="https://api.agipower.ai/v1", # [!code highlight]
    api_key="<你的 AGIPower_API_KEY>", # [!code highlight]
)

# 使用 Responses API 发起请求
response = client.responses.create(
    model="openai/gpt-5", # [!code highlight]
    input="生命的意义是什么？" # [!code highlight]
)

print(response.output_text)
```

```ts title="TypeScript"
import OpenAI from "openai";

const openai = new OpenAI({
  baseURL: "https://api.agipower.ai/v1", // [!code highlight]
  apiKey: "<你的 AGIPower_API_KEY>", // [!code highlight]

    model: "openai/gpt-5", // [!code highlight]
    input: "生命的意义是什么？", // [!code highlight]
  });

  console.log(response.output_text);
}

main();
```

```bash title="cURL"
curl https://api.agipower.ai/v1/responses \
  -H "Content-Type: application/json" \
```

### 协议三：Anthropic Messages

<Info title="兼容性说明">
AGIPower 完全支持 Anthropic Messages 协议，可以无缝集成到 Claude Code、Cursor 等工具中。

注意 Anthropic 协议的 `base_url` 为 `https://api.agipower.ai`，与 OpenAI 协议不同。
</Info>

<Info title="Anthropic 协议支持模型说明">
支持 Anthropic 协议的模型正在分批适配中，当前已支持的模型可以通过[官方模型列表](https://agipower.ai/models)筛选 Anthropic API Compatible 查看：
![anthropic-support](https://cdn.marmot-cloud.com/storage/agipower/2025/10/16/602FqX9/anthropic-support.png)
或者也可以通过[模型详情页](https://agipower.ai/anthropic/claude-haiku-4.5)查看：
![anthropic-support](https://cdn.marmot-cloud.com/storage/agipower/2025/10/16/I9JHS8b/detail-anthropic-support.png)
</Info>



```python title="Python"
from anthropic import Anthropic

# 1. 初始化 Anthropic 客户端
client = Anthropic(
    # 2. 将基础 URL 指向 AGIPower 端点
    base_url="https://api.agipower.ai", # [!code highlight]
    # 3. 替换为你从 AGIPower 获取的 API Key
    api_key="<你的 AGIPower_API_KEY>", # [!code highlight]
)

# 4. 发起 Messages 请求
message = client.messages.create(
    model="google/gemini-3.1-pro-preview", # [!code highlight]
    max_tokens=1024,
    messages=[
        {
            "role": "user",
            "content": "生命的意义是什么？"
        }
    ]
)

print(message.content[0].text)
```

```ts title="TypeScript"
import Anthropic from "@anthropic-ai/sdk";

```

### 协议四：Google Gemini

<Info title="兼容性说明">
AGIPower 支持 Google Gemini（Vertex AI）协议，可以使用 Google GenAI SDK 直接调用。

Base URL 为 `https://api.agipower.ai`，需要设置 `vertexai=True` 和自定义 `http_options`。
</Info>



```python title="Python"
from google import genai
from google.genai import types

# 1. 初始化 Google GenAI 客户端
client = genai.Client(
    # 2. 替换为你从 AGIPower 获取的 API Key
    api_key="<你的 AGIPower_API_KEY>", # [!code highlight]
    vertexai=True, # [!code highlight]
    # 3. 将 base_url 指向 AGIPower 端点
    http_options=types.HttpOptions(
        api_version='v1',
        base_url='https://api.agipower.ai' # [!code highlight]
    )
)

# 4. 发起 generateContent 请求
response = client.models.generate_content(
    model="google/gemini-3.1-pro-preview", # [!code highlight]
    contents="生命的意义是什么？"
)

print(response.text)
```

```ts title="TypeScript"
import { GoogleGenAI } from "@google/genai";

// 1. 初始化 Google GenAI 客户端
const ai = new GoogleGenAI({
  // 2. 替换为你从 AGIPower 获取的 API Key
  apiKey: "<你的 AGIPower_API_KEY>", // [!code highlight]
  vertexai: true, // [!code highlight]
  // 3. 将 base_url 指向 AGIPower 端点
  httpOptions: {
    apiVersion: "v1",
    baseUrl: "https://api.agipower.ai", // [!code highlight]
  },
});

async function main() {
  // 4. 发起 generateContent 请求
  const response = await ai.models.generateContent({
    model: "google/gemini-3.1-pro-preview", // [!code highlight]
    contents: "生命的意义是什么？",
  });

  console.log(response.text);
}

main();
```

## Platform API

AGIPower 提供一组 Platform Management API，让你能够通过编程方式查询账户用量、余额、订阅状态等信息，方便集成到你的监控和管理系统中。

<Warning title="⚠️ Management API Key">
Platform API 需要使用 **Management API Key** 鉴权（部分接口也支持普通 API Key）。前往 **[AGIPower 控制台 > Management](https://agipower.ai/platform/management)** 创建 Management API Key。
</Warning>

### 查询 Generation 详情

查询某次 API 调用的详细信息，包括 Token 用量、耗时、计费明细等。

```
GET https://api.agipower.ai/v1/management/generation?id=<generation_id>
```

每次 API 调用会返回一个 `generation_id`，使用该 ID 即可查询本次调用的完整信息：

```bash
curl https://api.agipower.ai/v1/management/generation?id=gen_01abc123def456 \
  -H "Authorization: Bearer $AGIPower_API_KEY"
```

返回示例（部分字段）：

```json
{
  "api": "chat.completions",
  "generationId": "gen_01abc123def456",
  "model": "openai/gpt-4o",
  "generationTime": 3200,
  "latency": 500,
  "nativeTokens": {
    "completion_tokens": 128,
    "prompt_tokens": 32,
    "total_tokens": 160
  },
  "usage": 0.0052
}
```

<Tip title="💡 计费信息延迟">
计费信息（如 `usage`、`ratingResponses`）需要在请求完成后 **3-5 分钟** 才能查询到。Token 计量信息则会随请求同步返回。
</Tip>

<Info title="📚 完整文档">
详细参数说明请参考 [Get Generation API 文档](/zh/api/platform/get-generation)。
</Info>
