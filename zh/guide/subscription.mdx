# 订阅制套餐

AGIPower Builder Plan 订阅制为个人开发者提供固定月费、可预测的 AI 模型调用方案,让您专注于编程和创作,无需担心每次 API 调用的费用。

## 什么是 Flows?

**Flow** 是 AGIPower 的复合计费单位，综合了 token 消耗量和 API 调用开销。你可以把它理解为一种**货币**——就像不同商品在美元中有不同的价格一样，不同的 AI 模型每次请求消耗的 Flows 数量也不同。Flow/USD 汇率可能会随时间波动，最新汇率始终在 [Pricing 页面](https://agipower.ai/pricing/subscription) 实时公布。

<Tip title="💡 Insider 会员 Flow 价值">
上述标准 Flow 价值适用于所有普通订阅用户。但对于保持**活跃、连续订阅**（不中断）的早期 **Insider 会员**，每个 Flow 将拥有更高的 USD 等价价值作为忠诚度奖励——这意味着你每消耗一个 Flow 能获得更多价值。
</Tip>

<Warning title="重要 — 滥用政策">
对于被发现违反 Builder Plan 服务条款的账号（如自动化滥用、资源囤积、未授权转售等），其有效 Flow 价值将被**降低至标准费率以下**。这意味着每个 Flow 的 USD 等价价值将减少。请合理使用你的订阅。
</Warning>

## 使用限制说明

<Warning title="⚠️ 重要提示">
订阅制套餐专为**个人开发、学习探索、Vibe Coding** 等非生产场景设计,请遵守以下使用规范:
</Warning>

### 速率限制

- **Rate Limit:** 10-15 RPM(每分钟请求数)
- **配额窗口:** 滚动 5 小时窗口内刷新
- **周限额:** 在每周滚动窗口内重置

### 适用场景

✅ **允许使用的场景:**

- 个人开发和学习
- Vibe Coding 和快速原型开发
- 技术探索和实验
- 个人项目和非商业化应用

❌ **禁止使用的场景:**

- 已上线的生产环境
- 商业化产品或服务
- 面向终端用户的应用
- 多账号池化轮询等滥用行为

<Tip title="💡 生产环境建议">
如果您的项目即将上线或已经商业化,请切换到 [Pay-As-You-Go](./pay-as-you-go) 按量付费方案,以获得:

- 更高的 SLA 保障
- 更稳定的服务质量
- 更灵活的扩展能力
- 专业的商业支持
</Tip>

## 如何订阅

### 第 1 步:了解套餐详情

访问 [AGIPower Pricing 页面](https://agipower.ai/pricing/subscription) 查看所有订阅套餐的详细信息和价格。

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/01/20/7shxqbP/dingyuezhi-jiage2-dapingbanben.png" alt="subscription-free" />

### 第 2 步:选择并订阅套餐

1. 在 Pricing 页面选择适合您的套餐
2. 点击对应套餐的 **"Get Max"** 或 **"Get Ultra"** 按钮(Pro 用户点击 **"Upgrade"**)
3. 完成支付流程


### 第 3 步:管理订阅和获取 API Key

订阅成功后,可访问 [订阅管理页面](https://agipower.ai/platform/subscription) :

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/01/20/4I6M3by/dingyuezhi-gerenguanliyemian2.png" alt="subscription-free" />

- 📊 **查看使用情况**
  - 当前 5 小时窗口的用量和剩余时间
  - 本周累计使用量统计
  - Flow 消耗明细

- 🔑 **获取订阅 API Key**
  - 生成专用于订阅制的 API Key
  - 管理和轮换现有 Key
  - 查看 Key 的最后使用时间

- 💳 **管理订阅**
  - 查看当前套餐信息
  - 升级或降级套餐
  - 查看账单历史

### Extra Usage - 超额自动切换

开启后,当 Builder Plan 订阅配额达到 5 小时或周限额时,自动切换至所选的 Pay As You Go Key 保证不中断使用;配额恢复后自动切回订阅 Key。

#### 💡 核心优势

- **无缝切换** - 配额用尽时自动切换到按量付费,无需手动更换 API Key
- **不中断工作流** - 不影响你的开发、编程或对话流程
- **自动恢复** - 订阅配额恢复后自动切换回订阅制计费
- **灵活可控** - 可随时启用或禁用此功能

#### 设置步骤

访问 [订阅管理页面](https://agipower.ai/platform/subscription),按照以下步骤配置 Extra Usage:

**步骤 1: 启用 Extra Usage**

在订阅管理页面找到 **Extra Usage** 部分,点击右侧的开关按钮启用此功能。

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/02/03/8Ed42do/extra-1.png" alt="Enable Extra Usage" />

**步骤 2: 选择 Pay As You Go API Key**

点击开关后会弹出 **"Select a Key"** 对话框,你可以:

- 从下拉列表中选择一个现有的 Pay As You Go API Key
- 或点击 **"Create new key"** 创建一个新的按量付费 API Key

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/02/03/iJZufsS/extra-2.png" alt="Select API Key" />

选择完成后点击 **"Select"** 按钮确认。

**步骤 3: 配置完成**

启用成功后,页面会显示你选择的 Pay As You Go API Key 的详细信息,包括:

- API Key 名称和密钥
- 启用状态
- 创建时间和最后使用时间
- 当前已使用金额
- 操作选项(可重新选择其他 Key)

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/02/03/7ivqY3m/extra-3.png" alt="Extra Usage Enabled" />

#### 工作原理

1. **正常使用** - 优先使用订阅制配额(Flows)
2. **配额用尽** - 当 5 小时窗口或周窗口配额达到上限时,自动切换到设定的 Pay As You Go API Key
3. **按量计费** - 此时的调用会从你的账户余额中扣费
4. **自动恢复** - 当订阅配额窗口重置后,自动切换回订阅制计费

<Tip title="💡 使用建议">

- 确保你的 Pay As You Go 账户有足够余额,以避免超额切换后因余额不足而中断服务
- 可以随时在订阅管理页面查看 Extra Usage 的使用情况和消费金额
- 如需更换备用 Key,点击 **"Reselect"** 按钮重新选择即可
</Tip>

<Warning title="⚠️ 计费提醒">
启用 Extra Usage 后,当订阅配额用尽时会自动从你的 Pay As You Go 余额中扣费。请注意监控账户余额,避免产生意外费用。
</Warning>

### 第 4 步:在开发工具中使用

获取订阅 API Key 后,即可在各类开发工具和应用中使用。

#### 与 Pay As You Go 使用方式一致

订阅制 API Key 的使用方式与 [Pay As You Go](./pay-as-you-go.md) 完全相同,支持 OpenAI SDK、Anthropic SDK 以及直接 HTTP 等调用。**唯一区别**是使用订阅制专属的 API Key(以 `sk-ss-v1-` 开头),调用时会自动从订阅配额中扣除,而不是从余额中扣费。

<Tip title="💡 API 调用示例">
完整的 API 调用代码示例请参考 [快速开始指南](./quickstart.md),只需将示例中的 API Key 替换为你的订阅制 API Key 即可。
</Tip>

## 在 Studio Chat 中使用订阅制

除了通过 API Key 在开发工具中使用,你还可以在 **[AGIPower Studio Chat](https://agipower.ai/chat)** 网页端直接使用订阅制配额进行对话。

### 切换计费方式

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/02/03/l48igiH/studio-chat-mode.png" alt="Studio Chat 计费模式切换" />

在 Studio Chat 页面,你可以在每个对话中灵活选择使用订阅制配额还是 Pay As You Go 余额:

如图，在对话设置中可以看到 **Billing Mode(计费模式)** 选项:

- **Subscription(订阅制)** - 使用订阅制配额(Flows),不消耗账户余额
- **Pay As You Go(按量付费)** - 从账户余额中按实际使用量扣费

<Tip title="💡 灵活切换">
你可以在不同对话中使用不同的计费模式。例如:

- 日常开发、学习、原型验证场景 → 使用 Subscription
- 生产环境测试、商业化项目验证 → 使用 Pay As You Go

两种模式可以随时切换,互不影响,让你在不同场景下都能选择最合适的计费方式。
</Tip>

<Warning title="⚠️ 配额共享">
Studio Chat 和 API 调用使用相同的订阅配额池。在 Studio Chat 中的对话会消耗订阅 Flows,请合理分配使用。
</Warning>

## 退款

如果你对当前订阅不满意或需要取消,可以在订阅管理页面发起退款申请。

### 退款资格

- ✅ **付费订阅**（Pro / Max / Ultra）可以在**订阅后 24 小时内**申请退款

<Warning title="⚠️ 退款时限">
退款申请必须在订阅成功后 **24 小时内**提交,逾期将无法发起退款。请在订阅后及时确认是否符合你的需求。
</Warning>

### 退款计算方式

退款金额按以下公式计算:

| 项目                      | 说明                              |
| ------------------------- | --------------------------------- |
| **已付总额**              | 当前订阅周期内已支付的全部费用    |
| **- 实际用量**            | 订阅期间已消耗的 API 调用等效价值 |
| **- 已退款**              | 之前已退款的金额（如有）          |
| **= 可退差额**            | 扣除用量和已退款后的剩余可退金额  |
| **- Stripe 手续费（5%）** | 支付平台收取的退款处理费用        |
| **= 最终退款金额**        | 实际退回到你支付方式的金额        |

<Warning title="⚠️ 无法退款的情况">
如果你的等效 API 使用价值已**超过订阅付款金额**,则该订阅无法退款。
</Warning>

### 退款流程

**第 1 步：发起退款**

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/03/20/fsjS4Sr/20260320154313.jpg" alt="发起退款" />

访问 [订阅管理页面](https://agipower.ai/platform/subscription),在账单历史（Billing History）区域点击 **"退款"** 按钮。

**第 2 步：确认退款详情**

<img src="https://cdn.marmot-cloud.com/storage/agipower/2026/03/20/7AmamOi/20260320153822.jpg" alt="确认退款详情" />

系统将显示退款预览,包括已付总额、实际用量、手续费和最终退款金额。确认无误后点击 **"提交退款"**。

**第 3 步：等待处理**

退款提交后,你可以在退款弹窗中查看处理状态:

| 状态      | 说明                                       |
| --------- | ------------------------------------------ |
| 🔵 处理中 | 退款正在处理中,将在 **1-3 个工作日**内退回 |
| 🟢 成功   | 退款已完成,资金已退回到你的支付方式        |
| 🔴 失败   | 退款无法处理,请联系客服获取帮助            |
| ⚪ 已取消 | 退款请求已被取消                           |

### 退款后的影响

退款成功后,请注意以下变化:

- 你的订阅将**立即降级为免费计划（Free）**
- 退款资金将在 **1-3 个工作日**内退回到原支付方式
- 待处理的 API 调用可能会影响最终退款金额

<Tip title="💡 需要帮助？">
如果你对退款计算有疑问或需要人工协助,请发送邮件至 [support@agipower.ai](mailto:support@agipower.ai)。
</Tip>

## 常见问题

### 订阅制和按量付费有什么区别?

| 特性             | 订阅制 (Builder Plan) | 按量付费 (Pay-As-You-Go) |
| ---------------- | --------------------- | ------------------------ |
| **计费方式**     | 固定月费              | 按实际使用量计费         |
| **适用场景**     | 个人开发、学习        | 生产环境、商业应用       |
| **费用可预测性** | ✅ 高(固定月费)       | ⚠️ 中(随用量波动)        |
| **速率限制**     | 10-15 RPM             | 更高,支持自定义          |
| **SLA 保障**     | 标准                  | 更高                     |
| **价格杠杆**     | ✅ 2.73-4.36 倍       | 标准 API 定价            |

### 什么时候配额会重置?

- **5 小时窗口:** 采用滚动窗口机制,从发送请求开始计量，每 5 小时重置一次
- **周限额:** 从发送请求开始计量，每满 7 天重置一次

您可以在[订阅管理页面](https://agipower.ai/platform/subscription)实时查看剩余配额和重置时间。

### 可以同时使用多个套餐吗?

不可以。一个账号只能同时激活一个订阅套餐。如需更高额度,请升级到 Max 或 Ultra 套餐。
