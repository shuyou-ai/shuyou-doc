# 下载发票

ShuYou 为所有充值和订阅付费用户提供完整的发票支持。本指南将帮助你了解如何填写发票信息、查看和下载发票，满足企业报销和财务审计需求。

## 如何下载发票

下载发票主要分为两个步骤：

1. **填写发票信息** - 在充值或订阅时输入开票资料
2. **下载发票** - 在账单历史中查看并下载对应的发票

## 第一步：填写发票信息

在进行充值或订阅付费时，你需要先填写发票信息。系统会根据你填写的资料生成发票。

### 在充值页面填写发票信息

当你进行 **Pay As You Go 充值**或**订阅套餐付费**时，在支付页面底部可以看到 **Invoice Details（发票详情）** 区域：

1. 点击 **Invoice Details** 区域的 **「Edit」** 按钮

<img src="https://cdn.marmot-cloud.com/storage/ShuYou/2026/01/21/wNHvxNw/ruhehuoqufapiao-fapiaoshenqingrukou.png" alt="发票信息入口" />

<br /><br />

2. 在弹出的表单中填写以下发票信息：

<img src="https://cdn.marmot-cloud.com/storage/ShuYou/2026/01/21/4G3z1Y6/ruhehuoqufapiao-tijiaofapiaoxinxi.png" alt="发票信息入口" />

| 字段                  | 说明                           | 必填  |
| --------------------- | ------------------------------ | ----- |
| **Full name**         | 发票抬头（个人姓名或公司名称） | ✅ 是 |
| **Country or region** | 国家或地区                     | ✅ 是 |
| **Address line 1**    | 详细地址（第一行）             | ✅ 是 |
| **Address line 2**    | 详细地址（第二行，可选）       | ⚪ 否 |
| **City**              | 城市                           | ✅ 是 |
| **State**             | 州/省                          | ✅ 是 |
| **ZIP code**          | 邮政编码                       | ✅ 是 |

3. 填写完成后，点击 **「Save billing info」** 保存发票信息
4. 继续完成支付流程

<Warning title="⚠️ 重要提示">
发票信息必须在支付前填写。如果支付时未填写发票信息，后续可能无法补开发票或需要联系客服处理。
</Warning>

### 修改已保存的发票信息

如果你已经保存过发票信息，下次充值或订阅时会自动使用之前保存的信息。如需修改：

1. 在支付页面点击 **Invoice Details** 区域的 **「Edit」** 按钮
2. 更新需要修改的字段
3. 点击 **「Save billing info」** 保存更新

系统会记住最新保存的发票信息，方便下次使用。

## 第二步：下载发票

完成充值或订阅付费后，你可以在账单历史中查看并下载对应的发票。

### 在 Pay As You Go 页面下载充值发票

如果你是通过 **Pay As You Go 充值**，可以在充值记录中下载发票：

1. 访问 **[Pay As You Go 页面](https://shuyou.ai/platform/pay-as-you-go)**
2. 在页面底部找到 **Transaction History（交易历史）** 区域
3. 在交易列表的 **Description / Invoice** 列中，找到对应充值记录
4. 点击 **「View Invoice」** 链接即可下载发票（PDF 格式）

<img src="https://cdn.marmot-cloud.com/storage/ShuYou/2026/01/21/csJydXK/ruhehuoqufapiao-anliangfufeijiemian.png" alt="Pay As You Go 交易历史" />

<Info title="💡 发票生成时间">
充值成功后，发票会立刻生成，点击 **View Invoice** 链接即可下载。

### 在 Subscription 页面下载订阅发票

如果你是通过 **订阅套餐付费**（Builder Plan），可以在订阅管理页面下载发票：

1. 访问 **[Subscription 页面](https://shuyou.ai/platform/subscription)**
2. 在页面中找到 **Billing History（账单历史）** 区域
3. 在账单列表的 **Invoice** 列中，找到对应订阅付费记录
4. 点击 **「View Invoice」** 链接即可下载发票（PDF 格式）

## 发票内容说明

ShuYou 发票包含以下信息：

- **发票编号**：唯一的发票识别码
- **开票日期**：发票生成日期
- **交易金额**：实际支付金额（美元）
- **发票抬头**：你填写的姓名或公司名称
- **账单地址**：你填写的完整地址信息

所有发票均为 **PDF 格式**，可直接打印或用于企业报销和财务审计。

## 常见问题

### 忘记填写发票信息怎么办？

如果支付时忘记填写发票信息，可以通过 Stripe 账单门户自行补充，无需联系客服：

**第 1 步：进入 Transaction History 页面**

在 Transaction History 中找到订单信息，点击提示中的 **"click here"** 链接，进入 Stripe 账单门户。

<img src="https://cdn.marmot-cloud.com/storage/ShuYou/2026/03/23/Xgn9aAa/20260323142642.jpg" alt="Transaction History 页面 - 点击 click here 进入账单门户" />

**第 2 步：更新账单信息**

在 Stripe 账单门户中，找到 **"账单和配送信息"** 区域，点击 **"更新信息"** 按钮，填写或修改你的发票信息（公司名称、邮件地址等）。更新后，后续生成的发票将自动使用新的信息。

### 如何修改已开具的发票信息？

已生成的发票无法直接修改。如果发票信息有误，请联系客服说明情况，并提供：

- 原发票的交易记录截图
- 正确的发票信息
- 修改原因说明

客服会根据实际情况协助处理。

### 发票可以用于企业报销吗？

可以。ShuYou 提供的 PDF 格式发票包含完整的交易信息和开票资料，符合企业报销和财务审计要求。

### 发票生成需要多久？

充值或订阅付费成功后，发票会立刻生成。如果超过 10 分钟仍未看到发票，请联系客服处理。

### 发票与合规支持

ShuYou 现已支持在线获取英文电子 Invoice。如涉及企业采购流程或特殊财务合规要求，请发送邮件至商务团队：

- **商务合作邮箱**：[support@shuyou.ai](mailto:support@shuyou.ai)
