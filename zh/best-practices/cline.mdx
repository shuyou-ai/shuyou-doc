# 通过 ShuYou 使用 Cline 指南

_Cline_ 是一款用于复杂工作的协作式编程智能体，通过与 ShuYou 的集成，您可以获得更多强大模型的支持，提升开发效率。

<Info title="兼容性说明">
ShuYou 完全支持 OpenAI API 协议，只需简单配置即可使用。

注意 OpenAI 协议的 base_url="https://api.shuyou.ai/v1"。
</Info>

## 安装 Cline

Cline 支持多种开发工具，如 Visual Studio Code, JetBrains IDEs 等，您可以根据自己的工具选择对应的安装方法.
具体可以参考 Cline 的官方文档

<Tip title="参考">
[Cline 官方文档](https://docs.cline.bot/getting-started/installing-cline)
</Tip>

## 配置 Cline

1. 点击 VSCode 左侧 Cline 图标, 打开 Cline 面板。
2. 点击面板右上角设置图标 ⚙️。
3. 在 API Configuration 标签页中:

- API Provider 选择 OpenAI Compatible
- Base URL 中填入 https://api.shuyou.ai/v1
- OpenAI Compatible API Key 填入 ShuYou API Key
- Model ID 填入 ShuYou 的模型 slug，如 anthropic/claude-3.7-sonnet

4. 点击右上角 Done 按钮完成配置。

## 开始使用

配置完成之后，即可开始使用 Cline。ShuYou 为 Cline 提供了丰富的模型支持，可以在我们的[官网模型列表](https://shuyou.ai/models)选择想要使用的模型填入 Model ID。

## 故障排除

### 常见问题解决

#### API Key 错误
**问题**：提示 API Key 无效或未授权

**解决方案**：

- 检查您输入的 ShuYou API Key 是否正确
- 确认 API Key 是否已激活且有足够余额
- 验证 API Key 格式是否以 `sk-` 开头
