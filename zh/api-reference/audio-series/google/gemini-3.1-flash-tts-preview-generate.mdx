## 文本（`input.prompt`）

必填。待合成文本（最多 **32,000** 字符）。可在文本中使用下列标记增强表现力：

| 标记 | 效果 |
| --- | --- |
| `[sigh]` | 叹气 |
| `[laughing]` | 笑声 |
| `[whispering]` | 低语 |
| `[shouting]` | 喊叫 |
| `[extremely fast]` | 极快语速 |

## 风格指令（`input.style_instructions`）

可选（最多 **1,000** 字符）。用自然语言描述语气、语速、口音与情感。默认值：`Say the following.`
