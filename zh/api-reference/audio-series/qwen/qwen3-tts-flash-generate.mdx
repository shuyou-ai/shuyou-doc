## 语种参数（`input.language`）

可选。指定合成音频的语种（对应 Qwen 文档字段 `language_type`）。默认 **`Auto`**。

| 取值 | 说明 |
| --- | --- |
| `Auto` | 适用于文本包含多种语言或语种不确定的场景。模型自动为不同语言片段匹配发音，但无法保证完全精准。 |
| `Chinese` | 单一中文文本，合成质量通常优于 `Auto` |
| `English` | 单一英语文本 |
| `German` | 单一德语文本 |
| `Italian` | 单一意大利语文本 |
| `Portuguese` | 单一葡萄牙语文本 |
| `Spanish` | 单一西班牙语文本 |
| `Japanese` | 单一日语文本 |
| `Korean` | 单一韩语文本 |
| `French` | 单一法语文本 |
| `Russian` | 单一俄语文本 |

<Tip>
文本为单一语种时，建议指定具体语种而非 `Auto`，以获得更好效果。
</Tip>
