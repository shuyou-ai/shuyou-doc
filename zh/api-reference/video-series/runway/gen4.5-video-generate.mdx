## 概览

**Runway Gen-4.5**（`gen4.5`）支持：

- **文生视频** — 仅传 `prompt`
- **图生视频** — 可选 `first_frame_url` 作为首帧

## 输入参数

| 字段 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `prompt` | 是 | — | 视频生成提示词 |
| `aspect_ratio` | 否 | `1:1` | `16:9`、`9:16`、`4:3`、`3:4`、`1:1`、`21:9` |
| `duration` | 否 | `5` | `5` 或 `10` 秒 |
| `first_frame_url` | 否 | — | 可选首帧图片 URL |
