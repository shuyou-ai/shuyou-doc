## 概览

**OpenAI Sora 2 Pro**（`sora-2-pro`）支持：

- **文生视频** — `prompt`（最多 **32,000** 字符）
- **参考引导生成** — 可选 `first_frame_url`

分辨率：**720P** 或 **1080P**。

## 输入参数

| 字段 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `prompt` | 是 | — | 最多 **32,000** 字符 |
| `aspect_ratio` | 否 | `16:9` | `16:9` 或 `9:16` |
| `resolution` | 否 | `720P` | `720P` 或 `1080P` |
| `duration` | 否 | `4` | `4`、`8`、`12` 秒 |
| `first_frame_url` | 否 | — | 可选参考 URL（如首帧图片） |
