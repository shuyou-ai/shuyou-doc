## 概览

**Kling V3**（`kling-v3-video`）支持：

- **文生视频** — 仅传 `prompt`
- **首尾帧** — `first_frame_url`、`last_frame_url`

## 输入参数

| 字段 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `prompt` | 是 | — | 最多 **2500** 字符 |
| `aspect_ratio` | 否 | `16:9` | `16:9`、`9:16`、`1:1` |
| `resolution` | 否 | `standard` | `standard` 或 `pro` |
| `duration` | 否 | `5` | `3`–`15`（秒） |
| `generate_audio` | 否 | `true` | 是否生成配套音频 |
| `first_frame_url` | 否 | — | 首帧图片 URL |
| `last_frame_url` | 否 | — | 尾帧图片 URL |
