---
title: 查询预测任务
description: 通过 GET /v1/predictions/{prediction_id} 使用 task_id 轮询异步预测任务。
openapi: zh/api-reference/task-management/get-prediction.json GET /v1/predictions/{prediction_id}
---

将 [Create Prediction](/zh/api-reference/task-management/create-prediction) 返回的 `data.task_id` 作为路径参数 `prediction_id` 传入。

## 响应字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `code` | string | 响应状态码，`"0"` 表示成功。 |
| `usage` | object | 计费摘要；任务处理中时可能尚未返回。 |
| `usage.upstream_cost` | number | 折扣前上游成本（USD）。 |
| `usage.discount` | number | 折扣金额（USD）。 |
| `usage.discount_percent` | number | 折扣比例（小数，如 `0.75` 表示 75 折）。 |
| `usage.cost` | number | 折扣后的最终计费金额（USD）。 |
| `data.task_id` | string | 异步任务 ID。 |
| `data.task_status` | string | 任务状态，如 `processing`、`success`、`failed`。 |
| `data.output` | array | 任务成功时的生成结果。 |
| `data.output[].type` | string | 输出媒体类型，如 `image`、`video`。 |
| `data.output[].image` | string | 当 `type` 为 `image` 时的图片 URL；链接可能有时效。 |
| `request_id` | string | 唯一请求 ID，用于排查与支持。 |
