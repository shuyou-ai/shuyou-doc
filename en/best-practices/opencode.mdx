# Guide to Using OpenCode with ShuYou

OpenCode is a terminal-first AI coding agent. Through its integration with ShuYou, you gain access to more powerful models to boost development efficiency.

## Install OpenCode

OpenCode offers multiple installation methods. Choose the one that best fits your system environment:



```bash title="Quick Install"
# Use the installation script (recommended)
curl -fsSL https://opencode.ai/install | bash
```

```bash title="npm/pnpm/yarn"
# Install with npm
npm i -g opencode-ai@latest

# Or install with pnpm
pnpm i -g opencode-ai@latest

# Or install with yarn
yarn global add opencode-ai@latest

# Or install with bun
bun i -g opencode-ai@latest
```

```bash title="macOS &amp; Linux"
# Install via Homebrew (macOS and Linux)
brew install sst/tap/opencode
```

```bash title="Windows"
# Install via Scoop
scoop bucket add extras
scoop install extras/opencode

# Or install via Chocolatey
choco install opencode
```

```bash title="Arch Linux"
# Install via paru
paru -S opencode-bin
```



<Tip title="More Installation Methods">
For more detailed installation options and instructions, please refer to the [OpenCode official documentation](https://opencode.ai/docs/#install)
</Tip>

## Configure OpenCode

### Sign in and configure ShuYou

After installation, you need to configure OpenCode to use the ShuYou API.

#### Step 1: Launch OpenCode and sign in

On first use, OpenCode will prompt you to sign in and configure:

```bash
opencode auth login
```

Select **ShuYou** in the provider selection screen:

&lt;div style="text-align: center;"&gt;
  <img src="https://cdn.marmot-cloud.com/storage/ShuYou/2025/10/30/PdPJcCv/opencode-login.jpg" alt="OpenCode Login" />
&lt;/div&gt;

<Warning title="Important Note">
Please ensure you select **ShuYou** as your API provider, not another vendor.
</Warning>

#### Step 2: Enter your ShuYou API Key

After selecting ShuYou, you will be prompted to enter your API Key. Please enter your ShuYou API Key (format `sk-xxx`):

&lt;div style="text-align: center;"&gt;
  <img src="https://cdn.marmot-cloud.com/storage/ShuYou/2025/10/30/nhMPtje/opencode-key.jpg" alt="Enter API Key" />
&lt;/div&gt;

<Tip title="Get API Key">
You can obtain or create your API Key in the [ShuYou Console](https://shuyou.ai/settings/keys).
</Tip>

#### Step 3: Choose a model

Once configured, use the `/models` command to view and select available models:

```bash
# Launch OpenCode
opencode

# List available models
/models
```

&lt;div style="text-align: center;"&gt;
  <img src="https://cdn.marmot-cloud.com/storage/ShuYou/2025/10/30/p7oX3KY/opencode-models.png" alt="Select Model" />
&lt;/div&gt;

## Troubleshooting

### Common Issues

#### API Key Error
**Issue**: API Key reported as invalid or unauthorized

**Solution**:

- Verify that the ShuYou API Key you entered is correct
- Confirm the API Key is active and has sufficient balance
- Check that the API Key format starts with `sk-`
- Re-run the `opencode` command and reconfigure
